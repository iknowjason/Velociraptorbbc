Examples

Verify if managed node is online:
aws ssm describe-instance-information \
    --output text --query "InstanceInformationList[*]"

Send a powershell command to EC2 Windows host:
aws ssm send-command \
    --instance-ids "i-0a797063a2ef5ab29" \
    --document-name "AWS-RunPowerShellScript" \
    --comment "IP config" \
    --parameters commands=hostname \
    --output text
